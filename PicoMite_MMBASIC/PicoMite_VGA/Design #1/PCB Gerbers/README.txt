These are the Gerber files which define the printed circuit board.
March 2022 Geoff Graham

The files are:
   PicoMiteVGA.GTL   Top Copper (GTL)
   PicoMiteVGA.GTS   Top Soldermask (GTS)
   PicoMiteVGA.GTO   Top Silkscreen (GTO)
   PicoMiteVGA.GBL   Bottom Copper (GBL)
   PicoMiteVGA.GBS   Bottom Soldermask (GBS)
   PicoMiteVGA.TXT   Drill File (2:5 leading)

Note that the drill file type is "2:5 leading"
